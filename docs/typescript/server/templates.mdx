# Project Templates

The `create-mcp-use-app` tool provides several project templates to help you get started quickly with different types of MCP servers. Each template is optimized for specific use cases and includes pre-configured examples.

<CodeGroup>
```bash npm
npx create-mcp-use-app@latest my-mcp-server
cd my-mcp-server
npm run dev
```

```bash pnpm
pnpm create mcp-use-app@latest my-mcp-server
cd my-mcp-server
pnpm run dev
```

```bash yarn
yarn create mcp-use-app@latest my-mcp-server
cd my-mcp-server
yarn dev
```
</CodeGroup>

<Tip>
To explore available templates and options, run:
```bash
npx create-mcp-use-app --help
```
</Tip>


## Available Templates

# Starter Template (Default)
 
The starter template is a comprehensive template that includes all the features of the UI Resource template, as well as the Apps SDK template.

**What's Included:**
- Full MCP server setup with tools, resources, and prompts
- React-based UI widgets with hot reload
- Kanban board example widget
- Development tools with auto-inspector
- Production build configuration

**Quick Start:**
```bash
npx create-mcp-use-app my-server --template starter
cd my-server
npm run dev
```
