The Supabase provider verifies Supabase-issued JWTs from your project. New Supabase projects use ES256 tokens with JWKS (no secret needed); older projects may still use HS256.

## Setup

### 1. Create a Supabase project

Go to the [Supabase Dashboard](https://app.supabase.com/) and create or select a project.

### 2. Get your project ID

Find your **Project ID** in **Project Settings → General**.

### 3. Check your JWT algorithm

Go to **Project Settings → API → JWT Settings**:

- **ES256** (new projects) — no secret needed, JWKS is used automatically
- **HS256** (legacy) — copy the **JWT Secret** and set it as `MCP_USE_OAUTH_SUPABASE_JWT_SECRET`

### 4. Configure authentication providers

In **Authentication → Providers**, enable the login methods you want (Email, Google, GitHub, etc.).

### 5. Environment variables

```bash
MCP_USE_OAUTH_SUPABASE_PROJECT_ID=your-project-id
MCP_USE_OAUTH_SUPABASE_JWT_SECRET=your-jwt-secret  # Only for HS256 (legacy)
```

## Configuration

<Tabs>
  <Tab title="ES256 (Recommended)">
    New Supabase projects use ES256 tokens with JWKS — no secret needed:

    ```typescript
    import { MCPServer, oauthSupabaseProvider } from 'mcp-use/server'

    // Zero-config: reads MCP_USE_OAUTH_SUPABASE_PROJECT_ID
    const server = new MCPServer({
      name: 'my-server',
      version: '1.0.0',
      oauth: oauthSupabaseProvider()
    })

    server.listen(3000)
    ```
  </Tab>
  <Tab title="HS256 (Legacy)">
    Older projects using HS256 require the JWT secret:

    ```typescript
    import { MCPServer, oauthSupabaseProvider } from 'mcp-use/server'

    const server = new MCPServer({
      name: 'my-server',
      version: '1.0.0',
      oauth: oauthSupabaseProvider({
        projectId: process.env.MCP_USE_OAUTH_SUPABASE_PROJECT_ID!,
        jwtSecret: process.env.MCP_USE_OAUTH_SUPABASE_JWT_SECRET!,
      })
    })

    server.listen(3000)
    ```
  </Tab>
</Tabs>

### All configuration options

```typescript
oauthSupabaseProvider({
  // Required (or set MCP_USE_OAUTH_SUPABASE_PROJECT_ID)
  projectId: 'your-project-id',

  // Required for HS256 tokens (or set MCP_USE_OAUTH_SUPABASE_JWT_SECRET)
  jwtSecret: process.env.MCP_USE_OAUTH_SUPABASE_JWT_SECRET,

  // Skip JWT verification — development only
  skipVerification: process.env.NODE_ENV !== 'production',

  // Override advertised scopes
  scopesSupported: ['openid', 'profile', 'email'],
})
```

## Accessing user info in tools

```typescript
server.tool(
  {
    name: 'get-user-info',
    description: 'Get authenticated user info',
  },
  async (_args, ctx) => ({
    userId: ctx.auth.user.userId,
    email: ctx.auth.user.email,
    name: ctx.auth.user.name,
    scopes: ctx.auth.scopes,
  })
)
```
