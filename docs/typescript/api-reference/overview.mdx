This is the reference for building MCP servers in TypeScript with `mcp-use`. It documents every public export, with exact signatures, parameters, return values, defaults, and a minimal example for each.

For task-based walkthroughs and concepts, use the [Docs](/typescript/server/index) tab. This section is the precise reference you cite when you already know what you are building.

## Install

```bash
npm install mcp-use
```

## Import paths

The server SDK ships from two entry points. Import server-side APIs from `mcp-use/server` and widget UI APIs from `mcp-use/react`.

| Import path | Use it to |
|---|---|
| `mcp-use/server` | Create the server, register tools, resources, and prompts, handle auth, sessions, middleware, and MCP Apps resources. |
| `mcp-use/react` | Build the React widget UI that renders inside MCP Apps and ChatGPT (the views your tools return). |

<Note>
Create a server with `new MCPServer({ ... })`. See the [MCPServer](/typescript/api-reference/server/mcp-server) reference for the constructor and all of its methods.
</Note>
